# Code Sync

Collaborative code editor with cursors and selections synced across all users in real time using [SpacetimeDB](https://spacetimedb.com).

## Instructions for Build and Use

Steps to build and/or run the software:

1. `bun install`
2. `spacetime generate`
3. `spacetime dev`

Instructions for using the software:

1. Open the hosted vite app in your browser (http://localhost:5173)
2. Open a second browser tab with the same url
3. Start editing the code in one tab, and see the changes and cursors in the other

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

- Bun (https://bun.sh)
- SpacetimeDB (https://spacetimedb.com)
